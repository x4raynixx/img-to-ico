# Convert to ICO

This tool allows you to quickly and easily convert a wide variety of popular image formats into ICO image type. It's fast, lightweight, and simple to use.

## Supported Image Types
- **PNG**
- **JPG**
- **JPEG**
- **WEBP**
- **BMP**
- **GIF**
- **TGA**
- **HDR**
- **PIC**
